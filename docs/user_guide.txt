1. 软件介绍
-----------
我的T(MyTarget)是一款简洁优雅的目标管理悬浮窗应用，专为提升工作效率和目标管理而设计。它可以帮助你高效管理和追踪本周、本月、本年的目标，支持中英文界面切换，并提供深色/浅色主题。

主要特点：
• 可拖动的悬浮窗口，随时置顶显示
• 按周/月/年分类管理目标
• 深色/浅色主题切换
• 中英文界面切换
• 目标优先级管理
• 智能最小化
• 励志语录滚动显示

2. 系统要求
-----------
• 操作系统
  - Windows 10/11 (64位)
  - 分辨率：1280x720 或更高

• 硬件要求
  - CPU：1.6GHz 或更快
  - 内存：2GB RAM
  - 硬盘空间：50MB 可用空间

• 软件环境
  - Python 3.8 或更高版本
  - pip（Python包管理器）
  - 网络连接（用于安装依赖）

3. 安装说明
-----------
源码运行方式：
1. 安装Python 3.x
   - 访问 Python 官网 (https://www.python.org) 下载并安装 Python 3.8 或更高版本
   - 安装时勾选"Add Python to PATH"选项

2. 下载源代码
   - 从 GitHub 下载项目源代码：https://github.com/junhe421/mytarget
   - 解压下载的文件到本地文件夹

3. 安装依赖
   - 打开命令提示符（CMD）或 PowerShell
   - 进入项目目录：cd 项目路径
   - 运行命令：pip install -r requirements.txt

4. 启动程序
   - 双击运行 start.bat
   或
   - 在命令行中运行：python main.py

注意事项：
• 确保电脑已正确安装 Python
• 确保网络连接正常以下载依赖包
• 如遇权限问题，请以管理员身份运行命令提示符

  - 待办事项提醒

5. 详细操作指南
--------------
4.1 基本操作
• 添加目标
  1. 选择时间范围（本周/本月/本年）
  2. 选择优先级（🔴高/🟡中/🟢低）
  3. 在输入框输入目标内容
  4. 点击"添加"按钮

• 编辑目标
  1. 右键点击目标
  2. 选择"编辑"
  3. 修改目标内容或优先级
  4. 点击"保存"

• 删除目标
  1. 右键点击目标
  2. 选择"删除"
  3. 确认删除

• 标记完成
  - 点击目标前的复选框即可标记/取消完成状态

4.2 窗口操作
• 移动窗口
  - 按住顶部蓝色区域拖动

• 切换主题
  - 点击右上角🌓按钮

• 切换语言
  - 点击右上角🌐按钮

• 最小化/还原
  - 窗口5秒无操作自动最小化为横幅
  - 双击横幅还原窗口

4.3 横幅模式
• 自动切换
  - 5秒无操作自动切换为横幅模式
  - 显示待办事项数量和励志语录

• 励志语录
  - 右键点击横幅可编辑语录
  - 支持添加自定义语录
  - 语录自动滚动显示

6. 数据管理
-----------
• 数据存储位置
  - 目标数据：data/goals.json
  - 励志语录：data/quotes.json
  - 设置文件：data/settings.json

• 数据备份
  1. 定期备份data文件夹下的所有文件
  2. 备份前建议先关闭程序
  3. 恢复时直接替换对应文件即可

• 数据迁移
  1. 复制整个data文件夹到新位置
  2. 确保文件夹结构保持不变
  3. 重启程序即可识别

7. 常见问题
-----------
Q: 如何备份数据？
A: 备份data文件夹下的goals.json和quotes.json文件即可。

Q: 窗口不见了怎么办？
A: 程序会自动保存在屏幕范围内，检查是否最小化或被其他窗口遮挡。

Q: 如何添加自定义语录？
A: 在横幅模式下右键点击，选择"添加语录"即可。

Q: 程序无法启动怎么办？
A: 1. 检查是否以管理员身份运行
   2. 确认是否安装了所需的.NET Framework
   3. 查看是否有杀毒软件拦截
   4. 检查data文件夹权限是否正确

Q: 数据文件损坏怎么办？
A: 1. 使用之前的备份恢复
   2. 如果没有备份，删除损坏的文件，程序会自动创建新的

Q: 如何修改默认语录？
A: 1. 在横幅模式下右键
   2. 选择"编辑语录"
   3. 直接修改或添加新语录
